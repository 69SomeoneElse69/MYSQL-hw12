База данных по мотивам сервиса YouTube

Сервис предназначен для предоставление огранизованного доступа к видео пользователей

Представленная база данных позволяет: 
1)организовать учетную запись пользователей;
2)позволяет огранизовать связь между учетной записью пользователя и созданими им каналами
3)позволяет огранизовать связь между учетной записью пользователя и 
  каналами созданными другими пользователями;
4)организовать загруженные видео по каналам пользователей
5)организовать хранение технической информации видео 
  (размер, субтитры, версии качества видео, ссылку на хостинг)
6)организовать обратную связь между пользователей в виде подписок на каналы
7)оргпнизовать обратную связь между пользователей в виде сообщений под видео
8)организовать обратную связь между пользователей в лайков для видео и коментариям к кему
